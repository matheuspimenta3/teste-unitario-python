# Teste Unitário com Python e PyUnit

## Descrição

Projeto desenvolvido para a aula prática de Engenharia de Software sobre teste unitário com Python e PyUnit.

A atividade consiste em criar uma calculadora simples e implementar testes unitários utilizando o módulo `unittest`.

## Estrutura do projeto

```text
teste-unitario-python/
│
├── calculadora.py
├── test_calculadora.py
├── README.md
└── README-IA.md
```

## Arquivos

| Arquivo | Descrição |
| :--- | :--- |
| `calculadora.py` | Contém as funções da calculadora |
| `test_calculadora.py` | Contém os testes unitários |
| `README.md` | Contém a descrição do projeto |
| `README-IA.md` | Contém o registro do uso de IA |

## Funções implementadas

| Função | Descrição |
| :--- | :--- |
| `somar(a, b)` | Retorna a soma de dois números |
| `subtrair(a, b)` | Retorna a subtração de dois números |
| `multiplicar(a, b)` | Retorna a multiplicação de dois números |
| `dividir(a, b)` | Retorna a divisão de dois números |
| `potencia(a, b)` | Retorna a potência de um número |
| `calcular_media(lista)` | Retorna a média dos números de uma lista |

## Como executar os testes

```bash
python -m unittest discover
```
